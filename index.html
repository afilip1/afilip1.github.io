my projects

Over the years I have worked on numerous little projects, most of which you can find on my GitHub. Here are some of the more notable ones:

Jute

(github)

A toy programming language made in ~1000 lines of C. Features basic type checking and native execution via LLVM. I had hoped to make a modern successor to C, but alas, writing a compiler in C turned out to be pretty painful. Well, at least it can handle Fizzbuzz.

Minekong

(github)

An aesthetically pleasing implementation of good old Minesweeper in React/Redux. Supports both mouse- and touch-optimized control schemes. Made with the purpose of learning React, but it's honestly quite addictive. Just don't ask about the name.

Seekr

(github)

A simple file browser written in Kotlin with TornadoFX. Supports navigation history, deleting and renaming files, updating on detecting external changes and other fancy things. Also a learning project, but it works well.
